# Parser XML

Parser for specyfic type of xml files. [TOOL]

## Usage

Install npm dependencies

```sh
 npm install 
```

**After the project has been kickstarted**

To start the development server

```sh
npm start
```

To build for production

```sh
npm run build
```

To preview the production build
```sh
npm run preview
```
